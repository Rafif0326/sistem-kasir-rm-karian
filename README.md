# Sistem Informasi Kasir dan Manajemen Pesanan Rumah Makan Karian

## Deskripsi

Sistem Informasi Kasir dan Manajemen Pesanan Berbasis Web Pada Rumah Makan Karian Menggunakan Metode Waterfall.

## Fitur Sistem

- Login Admin dan Kasir
- Dashboard Statistik
- Kelola Data Menu
- Transaksi Kasir
- Keranjang Belanja
- Cetak Struk Otomatis
- Manajemen Pesanan
- Detail Pesanan
- Laporan Penjualan
- Export Excel
- Grafik Penjualan
- Status Pesanan (Menunggu, Diproses, Dimasak, Selesai)

## Teknologi Yang Digunakan

- PHP Native
- MySQL
- Bootstrap 5
- Chart.js
- XAMPP

## Struktur Folder

```text
admin/
auth/
config/
menu/
transaksi/
assets/
database/
index.php
```

## Cara Menjalankan Sistem

1. Download source code
2. Letakkan folder pada:

```text
C:\xampp\htdocs\
```

3. Jalankan Apache dan MySQL pada XAMPP
4. Import database:

```text
database/db_rm_karian.sql
```

5. Buka browser:

```text
http://localhost/db_rm_karian
```
